# 高性能服务器网络编程(陈硕)

## 常用命令
### nc
命令参数:
- -l: 监听端口

### dd
命令参数：
- if: input file
- bs: bytes
- count: count
## TCP 性能检测
> 客户端: dd if=/dev/zero bs=1MB count=10000 | nc localhost 5001

> 服务端: nc -l 5001
