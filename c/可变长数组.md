## 可变长数组 char data[0]
在结构体最后放入char data[0]，可以让数组长度是可变的。
```c
struct buffer
{
  int data_len; // 长度
  char data[0]; // 起始地址
}
```

可以观察到char data[0]和char data[]都不占空间，而char* data，要占一个指针大小的空间。
data[0]和data[]的地址是紧接着结构体的地址，而char* data的地址不是。
