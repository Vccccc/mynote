# item.lua

## 函数

### Item.CreateItem(fullName, count, func)

#### 描述

根据 fullName 创建一个数量为 count 的 item 对象

#### 参数

- func : 回调函数，参数为物品数据

- count : 物品数

- fullName : item 名

#### 返回值

item 对象

### Item.CreateSlotItem(entity, tid, slot)

#### 描述

根据背包位置创建 item

#### 参数

- tid : tray id

- slot : 槽

- entity : 实体

#### 返回值

item 对象
