### Item.CreatItem(fullName, count, func)

#### 描述

根据 fullName 创建一个数量为 count 的 Item 对象。

#### 参数

- count : 物品数量

- func : 回调函数，参数为 Item 对象

- fullName : Item 名称

#### 返回值

返回 Item
