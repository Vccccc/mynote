# item.lua

## 函数

### init_obj(self, type, ...)

#### 描述

初始化 item

#### 参数

- type : ITEM_OBJ_TYPE_SETTLED、ITEM_OBJ_TYPE_ISOLATED

- self : 具体 item

- ... : item 数据

#### 返回值

null

### Item.CreatItem(fullName, count, func)

#### 描述

根据 fullName 创建一个数量为 count 的 Item 对象。

#### 参数

- func : 回调函数，参数为 Item 对象

- fullName : Item 名称

- count : 物品数量

#### 返回值

返回 Item
